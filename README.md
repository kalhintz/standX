## 다운로드

### Windows
[Releases](https://github.com/kalhintz/standx/releases)에서 최신 버전을 다운로드하세요.

1. `STANDX Trading Bot 1.0.0.exe` 다운로드
2. 원하는 폴더에 저장
3. 실행 (Windows Defender 경고 시 "자세한 정보" → "실행" 클릭)
4. 첫 실행 시 지갑 주소와 Private Key 입력

### 소스에서 빌드
```bash
git clone https://github.com/kalhintz/standX.git
cd standX
npm install
npm start          # 개발 모드
npm run build:win  # 빌드
```
